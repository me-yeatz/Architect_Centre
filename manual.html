Getting Started

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Internet connection for AI services
  • At least 4GB RAM recommended

Installation

  1. Clone the repository: git clone https://github.com/me-yeatz/Architect_Centre.git
  2. Navigate to the project directory: cd Architect_Centre
  3. Install dependencies: npm install
  4. Create a .env.local file and add your Gemini API key: GEMINI_API_KEY=your_api_key_here
  5. Start the application: npm run dev
  6. Visit http://localhost:3000 in your browser

Initial Setup

When you first launch the application, you'll see the default workflow. To customize it:

  1. Open the constants.ts file
  2. Modify the INITIAL_NODES, INITIAL_EDGES, and MOCK_CONTRACT to match your workflow
  3. Update the geminiService.ts file to customize AI prompts for your specific use case

Creating Workflows

Understanding Node Types

Trigger Nodes

Initiate workflow execution. Common triggers include:

  • New project creation
  • Milestone completion
  • File upload
  • Scheduled time

Action Nodes

Perform operations like:

  • AI content generation
  • Data processing
  • File manipulation
  • API calls

Condition Nodes

Make decisions based on data:

  • Check data values
  • Evaluate conditions
  • Route workflow differently

Response Nodes

Output results:

  • Send notifications
  • Save to storage
  • Send messages
  • Update records

Building Your First Workflow

  1. Add a Trigger: Click on the canvas to place a trigger node. This will be the starting point of your workflow.
  2. Add Actions: Place action nodes to perform tasks. You can add multiple actions that run in sequence or in parallel.
  3. Connect Nodes: Click and drag from the right port of one node to the left port of another to create connections.
  4. Configure Nodes: Double-click on nodes to configure their settings and parameters.
  5. Test Your Workflow: Use the "Run" button to execute your workflow and see the results.

Advanced Workflow Techniques

Parallel Processing

Connect one trigger to multiple action nodes to run tasks in parallel. This speeds up execution when tasks don't depend on each other.

Conditional Logic

Use condition nodes to create branching workflows. Based on certain criteria, your workflow can take different paths.

Looping Workflows

Create loops by connecting a later node back to an earlier one. Be careful to include exit conditions to prevent infinite loops.

AI Integration

Setting Up AI Services

AI Architect Centre uses Google's Gemini API for content generation. Follow these steps to set it up:

  1. Sign up for a Google AI Studio account at ai.google.dev
  2. Create a new project and obtain an API key
  3. Add the API key to your .env.local file as GEMINI_API_KEY
  4. Restart the application to load the new configuration

Customizing AI Prompts

You can customize the AI prompts in the services/geminiService.ts file:

// Example of customizing a prompt
const prompt = `As a marketing specialist for ${companyName}, create a compelling social media post about ${topic}. Tone: ${tone}. Length: ${length}.`;

Tips for effective prompts:

  • Be specific about the desired output format
  • Include context about your industry or company
  • Specify the tone and style you want
  • Define length constraints if needed
  • Provide examples when possible

Troubleshooting

Common Issues

AI Service Not Responding

Possible causes:

  • Invalid or expired API key
  • Network connectivity issues
  • Rate limiting from the AI service

Solutions:

  • Verify your API key in .env.local
  • Check your internet connection
  • Wait a few minutes if experiencing rate limits

Workflow Not Executing

Possible causes:

  • Circular dependencies in the workflow
  • Missing connections between nodes
  • Incorrect node configurations

Solutions:

  • Review your workflow for circular references
  • Ensure all nodes are properly connected
  • Double-check node configurations

Application Performance Issues

Possible causes:

  • Large workflow with many nodes
  • Browser memory limitations
  • Slow network connection

Solutions:

  • Break large workflows into smaller ones
  • Clear browser cache and restart
  • Try a different browser or device

Best Practices

Workflow Design

AI Integration
